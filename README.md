# 📖 介绍  

桂院校园导航是一款以地图为载体，提供桂林学院**校园地点的位置信息、导航，以及校园信息介绍**的小程序。  

旨在解决传统地图的校园标识不到位、地图形式低效单一、信息设计不够好等问题，为来桂院的新生和访客提供更加完美的出行体验。

# ⚡️ 功能

持续更新迭代中

## 地图相关
- ✅ 校园地点分类动态展示
- ✅ 地点选择与搜索
- ✅ 地点信息展示
- ✅ 路线规划

## 校园相关
- ✅ 校园信息展示
- ✅ 交通信息展示展示
- ✅ 新生攻略（还没做）
- ✅ 跳转至学校官微/招生小程序

## 在线管理
- ✅ 管理地点数据（地点类别、地点信息）
- ✅ 地点坐标获取（地图选点）
- ✅ 管理媒体文件（校园地图、封面、轮播图及视频）
